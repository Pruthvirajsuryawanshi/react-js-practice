## 🚀 Getting Started

For **CDN-based projects** (`Boxes`, `Social Buttons`, `Notification`, `congratsCardApp`, `q1`):
- Open the `index.html` file directly in your browser.

For **CRA / npm-based projects**:
```bash
cd <topic-folder>/<project-folder>
npm install
npm start
```
