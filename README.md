# ⚛️ React JS Practice

A collection of React.js practice projects covering core concepts from JSX fundamentals to component-based architecture and state management.

---

## 📂 Projects

### 1. [First React App](./firstReactApp/)
> Starter project bootstrapped with Create React App. Covers initial setup, project structure, and running a React development server.

**Concepts:** CRA setup, `npm start`, project scaffolding

---

### 2. [Congrats Card App](./congratsCardApp/)
> A congratulations card UI built using JSX elements and `ReactDOM.render()`.

**Concepts:** JSX, rendering elements, embedding expressions, image elements

---

### 3. [Boxes](./Boxes/)
> A responsive page displaying styled boxes with different background colors.

**Concepts:** JSX, CSS styling, responsive design, functional components

---

### 4. [Social Buttons](./Social%20Buttons/)
> Social interaction buttons (Like, Share, Comment) built with a reusable `Button` component using props.

**Concepts:** Functional components, props, component reusability

---

### 5. [Notification](./Notification/)
> A notifications app displaying different notification types (primary, success, warning, danger) with corresponding icons.

**Concepts:** Props, conditional styling, component composition, image rendering

---

### 6. [Reusable Banners App](./reusableBannersApp/)
> Renders a list of banner cards from data, demonstrating component reusability with dynamic props.

**Concepts:** Reusable components, `props`, list rendering, `src/components` structure

---

### 7. [Technology Cards](./technologyCards/)
> Displays technology cards (HTML, CSS, React, Bootstrap) with images and descriptions using a data-driven approach.

**Concepts:** Component composition, `cardsList` data, `key` prop, `CardItem` component

---

### 8. [Lists and Keys](./Lists-and-Keys/)
> Practice project focused on rendering dynamic lists and understanding the importance of keys in React.

**Concepts:** `Array.map()`, rendering lists, unique `key` attribute, preparing data

---

### 9. [Class Component and State](./class-component-and-state/)
> Introduction to class-based components, event handling in React, and managing component state with `setState()`.

**Concepts:** Class components, `this.state`, `setState()`, event handlers

---

### 10. [Super Over League (q1)](./q1/)
> A cricket-themed Super Over League page with team logos and scores.

**Concepts:** JSX, CSS layout, image elements, component styling

---

### 11. [Session Practices](./Session%20Practices/)
> In-session coding exercises covering React component creation and folder structure.

**Concepts:** Component fundamentals, project organization

---

## 🛠️ Tech Stack

- **React.js** — UI library
- **JavaScript (ES6+)** — Language
- **HTML5 & CSS3** — Markup & Styling
- **Create React App / CDN** — Build tooling

## 🚀 Getting Started

For **CDN-based projects** (`Boxes`, `Social Buttons`, `Notification`, `congratsCardApp`, `q1`):
- Open the `index.html` file directly in your browser.

For **CRA / npm-based projects** (`firstReactApp`, `reusableBannersApp`, `technologyCards`, `Lists-and-Keys`, `class-component-and-state`):
```bash
cd <project-folder>
npm install
npm start
```
